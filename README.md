# Is it Aromantic Spectrum Awareness Week?

SvelteKit site to show if it's [Aromantic Spectrum Awareness Week (ASAW)](https://www.arospecweek.org/).

## Developing

- Install deps with `pnpm install`
- Run the dev server with `pnpm run dev`
- Build the app with `pnpm run build`. You can preview the production build with `pnpm run preview`.
